Project: pbl-auth

System requirements:
- Node.js 20.x or newer
- npm 10.x or newer
- PostgreSQL database (for Prisma)

Install dependencies:
- npm install

Environment variables (.env):
- DATABASE_URL=
- NEXTAUTH_SECRET=
- JWT_SECRET=
- GOOGLE_CLIENT_ID=
- GOOGLE_CLIENT_SECRET=

Database setup:
- npx prisma generate
- npx prisma migrate dev

Run project:
- npm run dev

Build project:
- npm run build

Core npm packages (from package.json):
- next@16.1.6
- react@19.2.3
- react-dom@19.2.3
- next-auth@4.24.13
- @prisma/client@6.19.2
- prisma@6.19.2
- bcrypt@6.0.0
- jsonwebtoken@9.0.3
- zod@4.3.6
